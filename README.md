This project is a React Employee Directory application that fetches employee data 
from the public API JSONPlaceholder and displays it in responsive employee cards. 
The application uses React Hooks (useState and useEffect) to manage data fetching,
loading, and errors. Employee information is rendered dynamically using reusable
components and JavaScript array methods like .map(). It also includes interactive 
features such as a search bar for filtering employees by name or email, along with 
options to print and download employee cards, demonstrating modern React development 
and clean component-based architecture.
